PaperMC
The team of developers over at PaperMC is focused on improving the Minecraft ecosystem with faster and more secure software. LukeSMP uses both their server annd proxy software.

A fork of SpigotMC, Paper is a fast, lightweight, stable, open-source server software that retains support for the diverse selection of plugins programmed using the Spigot API while maintaining a snappy experience!

Although not necissary for LukeSMP, a seperate proxy server allows a place to offload tasks to free up resources. Thats where Velocity comes in

GeyserMC
One of the most popular server plugins today, Geyser creates a cross platform experience like none other, allowing Minecraft Bedrock Clients to connect to Minecraft Java servers!

Geyser is an open-source collaboration project with CubeCraft Games. Their goal is to allow Minecraft: Bedrock Edition users to join Minecraft: Java Edition servers as seamlessly as possible to allow for true cross-platform.

They are doing some amazing work in helping the Minecraft community come together. Thank you GeyserMC!

Minecraft BlueMap
Minecraft BlueMap is Google Earth for Minecraft! A program created by a team of volunteer developers, it's able to fetch the Minecraft world and then render it as a high 3d model. This 3d model is hosted on a webserver where you can view, zoom and the explore the world in 3 dimensions.

It does not end there though! Everynight, the pieces of the world where alterations occured within, are re-rendered, and then used to update the model on the website. This creates an expansive interactive experience that keeps you up-to-date about the current state of the world in near real-time!
