Software

Various programs and plugins used by LukeSMP

PaperMC

Turn your server up to 11 with PaperMC! Being a fork of SpigotMC, PaperMC is a fast, lightweight, stable, open-source server software that retains support for the diverse selection of plugins programmed using the Spigot API while also creating a snappy experience!

Currently used java plugins that allow incredible things
GeyserMC
Java - Bedrock Compatibility Layer

One of the most used plugins today, GeyserMC creates a cross platform experience like none other allowing Minecraft Bedrock Clients to connect to Minecraft Java servers!

DiscordSRV
Discord Hook

DiscordSRV allows the community to connect on a whole different level, building a bridge between the in-game Minecraft chat and Discord, connecting the online players to the offline players!

Learn More
Custom Maps
In-game Map Generator

A personally made plugin developed by the moderator (Luke), Custom Maps allows players to import links to images on the internet and render them onto maps in game with just a simple command!

Learn More
Dynmap

Dynmap is Google Maps for Minecraft! This plugin takes the Minecraft world and then renders it in thousands of tiles. The tiles are saved and then placed together on a website like a puzzle which you can then view, zoom, and explore the world with.

It does not end there though! When the world is altered by the placing/breaking of blocks, the tiles of which the changes occurred, are re-rendered, and then used to replace the old tile on the website. This creates a diverse interactive experience that keeps you up-to-date in near real-time!
